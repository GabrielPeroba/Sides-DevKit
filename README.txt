README - Instruções da Utilização da ferramenta SIDES (Sistema Inteligente de Detecçao de Eventos Sonoros)

- Mudar diretórios do código para o diretório de sons e imagens, respectivamente.

- Executar programa de interface juntamente ao programa principal para que haja comunicação.

- A detecção de sons começará após os gráficos de desempenho de classes serem exibidos.
